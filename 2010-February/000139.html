On Monday, 01. February 2010 10:01:49 Abhiram Kasina wrote:
> I am trying to run trousers daemon with tpm emulator. But when I am trying
> to start tcsd, it gives me an error. Can someone tell me whats wrong with
> what I am doing??
> 
> $ sudo insmod tpmd_dev/tpmd_dev.ko
> $ sudo tpmd
Please start the tpmd in foreground and debug mode
to facilitate the error identification:
sudo tpmd -df

Moreover, remember to add the clear argument if you
are calling the tpmd for the first time (see README):
sudo tpmd -df clear

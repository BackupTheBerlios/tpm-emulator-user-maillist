Hello all,

I recently downloaded the tpm emulator and installed it on my machine and
wanted to share the installation issues I had and how they can be fixed.

Problem 1
---

When running 'sudo make install' on my Ubuntu machine, I got the following
error:

make[1]: Entering directory `/home/akshay/tmp/tpm_emulator-0.5.1/tpmd'
install -m 755 -o tss -g tss -d /var/lib/tpm
install: invalid user `tss'

Solution:

run 'make' the following way:

sudo make TPMD_USER=root TPMD_GROUP=root install

I think that the documentation should include that the tpmd user either
needs to be created or modified during installation.

Problem 2
---

running 'modprobe' after a successfull install gives the following error*:

$ sudo modprobe tpmd_dev
**FATAL: Module tpmd_dev not found*

Solution:

run the following command before executing 'modprobe'

depmod -a

Again, i think this should be included as part of installation instructions
to make the emulator easier to install.


All in all installation was not very hard just some of the instructions were
missing :)

On Saturday, 14. October 2006 09:31, 0361024 at fudan.edu.cn wrote:
> I've got some questions about the non-volatile storage. It has not 
> been implemented yet according with the document on the main page. But 
> the storage root key of a TPM is supposed to be settled in NV storage. 
> So my question is that how can I create a storage root key? And what 
> about the secret?
The storage root key SRK is created as part of the take ownership
procedure (TPM_TakeOwnership). The NV storage you mentioned
is a TPM 1.2 feature that allows for the storage of external data
in the non volatile (and protected) storage area of the TPM.

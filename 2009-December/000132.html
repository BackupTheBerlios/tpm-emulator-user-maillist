On Thursday, 03. December 2009 00:24:33 Alexander Loukissas (aloukiss) wrote:
> I've successfully set up the tpm-emulator and I'm using the tpm-tools to
> interact with it. I'd like to know how I can reset/restore the TPM, so I
> can retake ownership of it.
The simplest way to reset the TPM emulator is to delete the state
file /var/lib/tpm/tpm_emulator-1.2.xxx and restart it in mode clear.
Of course you can also restore a previous state by replacing the
file with a backup copy.

On Wednesday, 24. September 2008 09:54:22 Shaz wrote:
> I am using tpm emulator 0.5 and trousers 0.3.1-r0. After "tpmd -fd" is
> run and waiting for connection I run "tcsd -f" but it gives the
> following error:
> 
>        TCSD TDDL ERROR: Could not find a device to open!
> 
> "tpmd -df" is listening and waiting for connections and "/var/tpm" has
> the created socket for tpmd. What can be the possible reason for this error?
Trousers uses the device /dev/tpm (or /dev/tpm0) for
communicating with the TPM. Make sure that the tpmd_dev kernel
module is loaded and that tcsd has the rights to access it (r+w).

Regards,
Mario

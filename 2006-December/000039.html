The __stack_chk_fail warning is not related to the stack issue of FC5
that is responsible for the system freeze. In order to  get rid of the
warning, just append the following code at the end of linux_module.c:

void __stack_chk_fail(void)
{
      error("stack-protector: stack is corrupted\n");
}

> Hi there!
>
> I've created a virtual TPM device for virtualbox which emulates
> a TIS device to guest system and uses the TPM Emulator on the
> other side.
>
> The code is available at:
>         http://www.trust.rub.de/projects/vTPM/
>
> Unfortunately, the module can't (yet) be built by itself, but
> must be compiled into virtualbox (b/c i haven't yet figured out
> how to build devices as modules).
> Building the code requires some (very crude) patching and has
> no UI or commandline configuration hooks (yet) - so it's all
> a bit hacky but once set up, the module runs stable and
> functional ;-)

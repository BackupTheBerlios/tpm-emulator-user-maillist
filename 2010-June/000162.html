> i would like to know how to reset pcrs after having extended them, i
> mean, even a reboot doesn't seem to work, and i haven't founud anywhere
> else how to reset them, is it a problem in my installation maybe? or is
> it simply that i haven't read something?
By default the TPM emulator is started in "persistent"
mode which means that (almost) all internal data is
restored. To start it in "clear" mode simply specify
"clear" as a command line parameter (e.g., tpmd -df clear).
An entirely clean restart can be achieved by removing
the persistent data before restart: rm -rf /var/lib/tpm.

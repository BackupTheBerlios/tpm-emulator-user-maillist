Dear all,

I am happy to announce the latest release of the TPM Emulator
(see https://developer.berlios.de/project/showfiles.php?group_id=2491).
which is the first release to fully implement all 1.2 TPM features
and to pass all respective tests of the trousers test suite
(http://trousers.cvs.sourceforge.net/viewvc/trousers/testsuite/ ).

In view of the limited number of platforms and applications that
we can use to test the emulator, we heavily depend on the help
of others to further improve the quality and portability of the
emulator. I therefore kindly ask you to test this release with all
additional tools and test cases available to you and to report any
bugs or compatibility issues to me. We further welcome any
information on (un)successful attempts to run/port the emulator
to other platforms than Linux. Thank you very much for your help!

Kind regards,
Mario

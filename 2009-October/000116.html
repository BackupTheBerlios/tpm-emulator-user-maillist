What about the tpmd_dev kernel module -- has it been loaded?
If not please do so, e.g. by calling

# insmod tpmd_dev/tpmd_dev.ko

and check whether the tpm device was properly set up

# ls -l /dev/tpm*

Should the device files not be available although the module
is loaded, you can try to manually create them:

# mknod /dev/tpm c 10 224
# ln -s /dev/tpm /dev/tpm0
# chown tss.tss /dev/tpm
# chmod 664 /dev/tpm

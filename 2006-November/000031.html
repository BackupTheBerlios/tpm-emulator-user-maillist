The default stack size in the original Fedora Core 5 kernels is only 4K which is not
enough for some GNU MP functions. If you do not want to recompile a new kernel
by yourself, you can chose one of those provided at
http://www.linuxant.com/driverloader/wlan/full/downloads-fc5-kernel-i686.php
Note that you have to install both, the kernel image as well as the kernel sources!
